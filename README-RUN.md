## How to run the app
### IDE tool
Simply run the `DrawingApplication.java#main` method in your favorite IDE tool

### Package and run
#### Generate a JAR
Command line :
```
mvn clean install
```
JAR location :  `target/drawingprogram-1.0.0.jar`

#### Execute the JAR
Command line :
```
java -jar drawingprogram-1.0.0.jar
```
